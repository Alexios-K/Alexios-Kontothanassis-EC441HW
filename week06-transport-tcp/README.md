# Week 6: Transport Layer & TCP

**Topic:** Transport Layer (TCP, UDP, QUIC)
**Type:** Report
**Description:** This report (`report.md`) compares the two primary Transport Layer protocols: TCP and UDP. It covers TCP's connection-oriented features such as the 3-way handshake, reliability mechanisms, and flow/congestion control, contrasting them with UDP's lightweight, connectionless approach. The report also introduces QUIC, a modern UDP-based protocol designed to reduce latency and improve connection migration for modern web applications.
