# Week 7: Reliable Data Transfer

**Topic:** Reliable data transfer (Stop-and-Wait, sliding window, GBN, SR)
**Type:** Problem
**Description:** This week's problem set covers Reliable Data Transfer (RDT) mechanisms at the Transport layer. The `problem.md` file analyzes the efficiency of Stop-and-Wait over a high-latency satellite link and compares the retransmission behaviors of Go-Back-N (GBN) and Selective Repeat (SR) protocols.
