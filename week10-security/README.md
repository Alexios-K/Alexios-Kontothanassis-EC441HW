# Week 10: Security

**Topic:** Security (cryptography, protocols)
**Type:** Report
**Description:** This report (`report.md`) explores the mechanics of modern encryption, specifically focusing on SSL/TLS. It breaks down the differences between Asymmetric and Symmetric encryption and explains how the TLS 1.3 handshake uses a hybrid approach (like Ephemeral Diffie-Hellman) to secure modern application layer communications.
