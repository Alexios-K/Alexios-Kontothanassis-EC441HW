# Week 3: Multiple Access & Ethernet

**Topic:** Multiple access (ALOHA, CSMA), Ethernet (addressing, switching, ARP)
**Type:** Report
**Description:** 
This report (`report.md`) provides an in-depth look at Carrier Sense Multiple Access with Collision Detection (CSMA/CD), a foundational protocol for managing shared media in early Ethernet networks. It covers the mechanics of CSMA/CD, including Carrier Sense, Multiple Access, and Collision Detection, as well as the "Jam" and "Backoff" process used to resolve collisions. The report also discusses the evolution to modern Switched Ethernet, where full-duplex communication and micro-segmentation have largely rendered CSMA/CD obsolete. Finally, it draws a comparison to wireless networks and the use of CSMA/CA (Collision Avoidance) in Wi-Fi (802.11).
